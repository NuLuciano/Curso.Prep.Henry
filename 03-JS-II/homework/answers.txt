for: Se utiliza para hacer bucles que tienen como objetivo realizar muchas tareas, que si se tendrían que codear 
habría que escribir mucho. Su función principal es la de no repetir codigo. Pueden ser infinitos o finitos según 
sea requerido. La declaración for esta compuesta por una variable, una condición y un repetidor. La variable es 
el objeto que se usará, la condición es lo que dará la ruta de escape en caso que se tenga que cortar el bucle y 
el repetidor es lo que hará que la operación itere cuantas veces sea necesario.

&&: Es un operador lógico que equivale a AND o Y, sirve para verificar si dos expresiones cumplen una condición 
y si lo hacen dará un true, pero si una de las dos expresiones no cumple la condición entonces devolverá un false, 
además si ninguna cumple la condición entonces tambien dará false.
Por ejemplo: if ( 1 + 1 === 2 && 2 + 2 === 4 ) ambas expresiones son true y se ejecuta el if.
if ( 1 + 1 === 2 && 2 + 2 === 5 ) una expresión es true pero la otra es false y no se ejecuta el if.

||: Es un operador lógico como el &&, equivale a OR, O. A diferencia del AND solo necesita que una de las expresiones
cumpla la condición para que tire un true, y solo será false si ninguna expresión cumple la condición.

!: Equivale a NOT, NO y devuelve el valor opuesto de lo que se le pasa. Por ejemplo !true será false, ya que false
es el valor booleano opuesto de true. !(1 + 1 === 2) devolverá un false.
